ASCOM Switch Driver For Govee Devices

ASCOM switch driver to control simple Gove switch devices like the model H5083 smart plugs and similar models.  The driver will store up to f4 devices that can be displayed in Nina and other programs using ASCOM drivers.
You will need to get an API Key from Govee.com (free for non-commercial uses, can obtain via smart device app or online from govee.com)).  You will also need the switches' MAC addresses, and device type.  NOTE: The MAC address listed in Govee's provided app does not list a MAC addresses compatible with the API key method of controlling their devices.  I can't find any info related to this online or in their help files. Click here for the MAC discovery tool's download link = https://ccdastro.com/GoveeGo.zip
