ASCOM Switch Driver For Govee Devices

ASCOM switch driver to control simple Govee switch devices like the model H5083 smart plugs and similar models.  The driver will store up to 4 devices that can be controlled in NINA and other programs using ASCOM drivers.
You will need an API Key from https://Govee.com (free for non-commercial uses, can obtain via smart device app or online).  You will also need the switches' MAC addresses, and device type.  NOTE: The MAC address listed in Govee's provided app does not list a MAC addresses compatible with the API key method of controlling their devices.  Go [here](https://ccdastro.com/GoveeGo.zip) for a MAC discovery tool installer that will properly gather required information from the Govee devices you have setup and are listed on your Govee smart devices.
