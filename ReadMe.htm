ASCOM Switch Driver (C#)



You have just installed an ASCOM SWitch driver.

This version currently only allows the control of 1 switch

  • You will need to get an API Key from Govee.com (free for non-commercial uses), then have the correct switch's MAC and device type (can be retreived using my standalone Windows app, code is available if the adventurous want to add to the switch driver). ALERT: The MAC address listed in Govee's provided at app stores does not list a MAC compatible with the API key method of controlling their devices. I can't find any info related to this online or in their help files. Click here for the MAC discovery tool's download link.
